## Code Hike + Remotion

Blog post: [From Markdown to Video](https://codehike.org/blog/remotion)

Code Hike docs: https://codehike.org/docs/  
Remotion docs: https://www.remotion.dev/docs/

### Clone this example

```bash
$ npx degit code-hike/examples/with-remotion my-video
$ cd my-video
$ npm install
$ npm run start
```
